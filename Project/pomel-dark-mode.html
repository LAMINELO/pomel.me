What is it ?


-Pomel is an app made to help people use and organize their time correctly.

-Pomel Can be used for homeworks ,studies and any type of tasks.

-We all hate doing work but with Pomel it will be more easier and actually it will be fun.


How it works ?

Pomel uses 'Pomodoro technique' Which works like next:
  • Start the timer and do the wanted task for 25 min.

  • Take a rest for 5 min.

  • Re-do the steps till we complete our goal.

Advices


Here some advices for better results:
  • Try to sit in a calm place which help to relax.

  • Keep away any type of distractions. 'The phone for example' But if you had to bring it turn the airplane mode.

  • Bring a drink you prefer.'coffee, juice...etc'

  • Play some music.
